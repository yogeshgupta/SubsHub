										-----Spectrum V1.1 Read Me ----
About~

Thanks for download SubsHub .This is the first release and hence the basic.
You can download the subtitles of any movie or sitcom by the use of SubHub in any language.Subtitle will be saved in the same
directory as of the movie and with the same name as of the movie

Requirments~
Dot Net Framework 4.0

How To Use~

SubsHub is installed in C:/Program Files/OpenHub Softwares/SubsHub by default.Besides that you can also found a shortcut to 
the subshub on your Desktop and program menu in the latest version of SubHub 

SubsHub uses simple UI making it easy even for novice computer user.
Select the option of folder if you want to select a folder/directory/drive,where multiple video files are present and you want to get the 
subtitles for all the video files in a single instance.It is to be noted that the subhub will get you the subtitles of video files also present in the
subdirectory of the folder selected.
If you want to get a subtitle for a single file select the file option ,even if you will not by default you will be able to
select a file .You will need to go inside the specific directory where 
your video file is present and select that video file .

How To select the language of your subtitles~

Latest version of SubsHub provide support to download subtitles in multiple languages.English is selected as the default language for subtitles.
To selects the language of your choice,go to preferense in the menu bar select subtitle language.
A new window will appear where you can select the language of subtitle of your choice.After selecting the language in which you want your 
subtitle in press select.Now browse the directory or files as mentioned above and enjoy the hackle free downloading of the subtitles in your language :)

Batch subtitles download over network location and SmartDownload option added~

Now you can download multiple subtitles( or batch download) using Subshub. A feature of smart download is also added which if selected will not download 
the subtitles for the video files which already have subtitles availaible in the directory. A clear screen button is also given to clear the console which
displays the downloading details of subtitles. All these updates are contributed by Ajay Singh( ajay0055@gmail.com ) a tech blogger(Techovity.com).
